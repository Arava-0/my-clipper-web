# my-clipper-web

Site de présentation de [My Clipper](https://github.com/Arava-0/my-clipper), un outil de découpage vidéo sans perte pour Windows.

Construit avec React + Vite + Tailwind CSS v4. Déployable via Docker.

---

Presentation website for [My Clipper](https://github.com/Arava-0/my-clipper), a lossless video clipping tool for Windows.

Built with React + Vite + Tailwind CSS v4. Deployable via Docker.

---

## Dev

```bash
npm install
npm run dev
```

## Deploy

```bash
make setup   # generate .env from .env.example
make up      # build + start
```
